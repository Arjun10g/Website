Current Work

Active Research

SEM Psychometrics
Influential Functions in Structural Equation Modelling

Developing influence diagnostics for SEM using influence function theory - identifying observations that disproportionately affect parameter estimates, model fit indices, and path coefficients in latent variable models.

Statistics Non-Parametric
The SaddlePoint Divergence Metric (SPDM)

A non-parametric probability density-based framework for quantifying distributional differences - leveraging saddlepoint approximations to construct a novel divergence metric with strong theoretical guarantees. Manuscript under review.

Under Review · Communications in Statistics - Simulation and Computation
SEM Optimisation
Gradient-Based Trajectory Informed Optimiser for SEM

Adapting gradient-based trajectory optimisation methods to structural equation modelling - replacing traditional ML/GLS estimators with trajectory-informed gradient descent to improve convergence in complex, high-dimensional SEM specifications.

ML Rashomon
A Rashomon Diagnostic Framework for Model Disagreement in Psychological ML

A diagnostic framework for quantifying and interpreting disagreement among near-optimal ("Rashomon") models in psychological machine learning. Demonstrated on NHANES depression prediction with logistic regression, Random Forest, and XGBoost to expose hidden instability in model-based inference.

Under Review · Psychological Methods (MET-2026-0611)
MLM Simulation
Ordinal Predictors in Multilevel Models

Comprehensive SimDesign-based simulation comparing eight methods for handling ordinal predictors in two-level random-intercept models across a 3^6 factorial design (729 conditions, 1,000 reps each). Introduces two novel methods (LCP and IC) and provides practical recommendations for applied researchers.

Under Review · Psychological Methods (MET-2026-0590)

Clinical RAG pgvector FastAPI · HTMX
rag-psych — Clinical Retrieval-Augmented Generation

Hybrid retrieval over MTSamples + PubMed + ICD-11. Three retrievers per source (dense + BM25 + rare-token lexical) → RRF fusion → cross-encoder rerank → custom rule-based NegEx negation filter → Anthropic generation with forced citations validated server-side. 100% citation validity on a 16-query hand-labelled eval; full audit logging; refusal short-circuit when retrieval confidence falls below threshold.

Try it live ↗  ·  github
DuckDB-WASM React · TypeScript Client-only
Data Cleaning Studio — in-browser reproducible data-cleaning

Every user action is a recipe step, so the current table is always replay(raw, recipe) — undo/redo, reproducibility, and one-click export of the recipe to R (dplyr/tidyr), Python (pandas), or SQL. Import XLSX/CSV/Parquet/Arrow with a sheet picker; promote rows to headers; regex-aware filter builder; type-aware transforms (z, Box-Cox, RankINT); distributions (hist, KDE, ECDF, QQ, beeswarm); missingness (φ co-occurrence, UpSet); joins/unions/pivots. Everything runs client-side via DuckDB-WASM — no backend, no data upload.
